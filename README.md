# Customer Reviews Sentiment Analysis (ML + Power BI)

End-to-end NLP + ML project to analyze Amazon reviews and generate business insights.

## Key Highlights
- ~100K reviews analyzed
- TF-IDF + Logistic Regression
- Accuracy: ~84.5%
- Power BI Dashboard for insights

## Tech Stack
Python, NLP, Scikit-learn, Power BI
